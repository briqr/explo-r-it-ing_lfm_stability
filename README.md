# Exploring and Exploiting Stability in Latent Flow Matching

The repository for our paper
**Exploring and Exploiting Stability in Latent Flow Matching**  
[arXiv](https://arxiv.org/abs/2605.08398), accepted at ICML 2026.

Code and pretrained models are being released.

## Models

Pretrained models are now available on Hugging Face:

[https://huggingface.co/briqr/exploring-exploiting-stability-lfm](https://huggingface.co/briqr/exploring-exploiting-stability-lfm)
