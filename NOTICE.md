# Notice and Third-Party Code

This repository contains the source code of our paper

**Exploring and Exploiting Stability in Latent Flow Matching**

Some components are adapted from existing open-source projects:

## flex_gen

The `src/autoencoders/` package is adapted from the flex_gen framework:

https://github.com/HelmholtzAI-FZJ/flex_gen

## DiT

Some model and training components are adapted from the official DiT implementation:

https://github.com/facebookresearch/DiT

## SiT

Some flow-matching, transport, and sampling components are adapted from SiT:

https://github.com/willisma/SiT

## torch-kmeans

Some clustering utilities are adapted from torch-kmeans:

https://github.com/jokofa/torch_kmeans

We thank these projects' authors for making their libraries publicly available. Please refer to the respective repositories for their original licenses.
